RGPD & Protection des données

Gestion des données personnelles des collaborateurs dans l'Active Directory

Dans le cadre de mon support IT chez Sodiaal, je manipule des données personnelles dans l'Active Directory (nom, prénom, e-mail, service, poste), dans le respect du RGPD.

Suppression des comptes sortants

Départ d'un collaborateur → compte AD désactivé immédiatement, puis supprimé après délai défini. Données associées purgées (boîte mail, dossiers) selon la politique de rétention → droit à l'effacement garanti.

Droits d'accès limités

Permissions attribuées selon le principe du moindre privilège : accès uniquement aux ressources nécessaires. Groupes de sécurité AD structurés par service et revus périodiquement.

Principe de minimisation

Seules les infos strictement nécessaires sont collectées dans l'AD (identité, fonction, rattachement). Aucune donnée sensible (n° SS, bancaire) n'est stockée.

Conformité RGPD

Base légale identifiée (contrat de travail) · Registre des traitements tenu par le DPO · Durées de conservation définies · Collaborateurs informés via la charte informatique.
