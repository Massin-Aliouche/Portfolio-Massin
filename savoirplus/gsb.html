Le projet GSB

🎯 Objectifs principaux

  • Mettre en place un serveur de virtualisation Proxmox
  • Déployer un Active Directory pour la gestion centralisée des utilisateurs
  • Installer un outil ITSM (GLPI) pour la gestion du parc et du support
  • Mettre en œuvre des bonnes pratiques d'administration système et réseau
  • Travailler en autonomie sur des problématiques réelles (réseau, services, dépannage)
Télécharger le sujet

Travaux réalisés

1) Virtualisation

  • Installation et configuration de Proxmox VE
  • Création et gestion de machines virtuelles Windows Server
  • Configuration réseau (bridge, accès web Proxmox)

2) Active Directory

  • Installation d'un contrôleur de domaine Windows Server
  • Création et organisation des utilisateurs et groupes
  • Groupes (administration, utilisateurs, services)
  • Mise en place d'une structure logique (OU)

3) Compétences mobilisées

Administration Windows Server
Virtualisation avec Proxmox
Réseaux (IP, accès, services)
Gestion des identités (AD)
Déploiement de services web
Méthodologie SISR

4) Résultat

Une infrastructure fonctionnelle, virtualisée et sujette à évolution, permettant :

  • La gestion centralisée des utilisateurs
  • Le suivi du parc informatique
  • La gestion des incidents via GLPI
  • Une base solide pour l'ajout de nouveaux services (supervision, sécurité, GPO…)

5) Schéma réseau de l'infrastructure
